# BNB

BNB is a full-stack web application inspired by Airbnb.
It allows users to create property listings, browse available stays, make bookings, cancel reservations, and rate properties.

The project is built to practice end-to-end full-stack development, including authentication, CRUD operations, image uploads, and deployment.

---

## What the Project Does

* User authentication (signup, login, logout)
* Create, edit, and delete property listings
* Upload images for listings
* Book properties with check-in and check-out dates
* Cancel bookings
* Rate listings
* Search listings by city, landmark, or title

## Technologies Used

### Frontend

* React (Vite)
* React Router DOM
* Tailwind CSS
* Axios
* Context API

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT (authentication using HTTP-only cookies)
* bcryptjs

### Image & Data Storage

* Cloudinary (image hosting)
* MongoDB Atlas (database)

### Deployment

* GitHub
* Render (frontend and backend)
